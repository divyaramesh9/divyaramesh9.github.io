Semantic Role Labelling with BERT

Approaching Semantic Role Labelling as aB-I-O sequence tagging problem, I constructed a PropBank-style semantic role labeling system using PyTorch and HuggingFace Transformers. I fine-tuned BERT for token-level classification by adding a custom classification head to predict semantic roles for each token in a sentence.

To improve argument identification accuracy, I leveraged BERT's segment embeddings to encode predicate positions, allowing the model to better understand the relationship between predicates and their arguments when identifying semantic roles in natural language text.

Python PyTorch HuggingFace

Conditioned LSTM Image Captioning Model

In the pursuit of improving Image caption generation through solely greedy search, I constructed an image captioning pipeline using PyTorch and ResNet, where ResNet extracts visual features and a conditioned LSTM generates descriptive captions as sequences. To produce higher-quality captions, I integrated beam search decoding, which explores multiple caption possibilities before selecting the best one.

Through multi-modal feature conditioning that better aligns visual and textual information, I improved image-caption relevance and fluency on the Flickr dataset by approximately 5-10% as measured by the CIDEr metric, demonstrating more accurate and natural-sounding descriptions of images.

PyTorch ResNet

Network Chat Client

I implemented an interactive command-line chat client in C using TCP sockets, handling user input validation, message framing, and concurrent I/O operations. I built a custom message framing protocol using length-prefixes over sockets to ensure correct message boundaries and support arbitrary message sizes, preventing data corruption when multiple messages are sent in quick succession.

To enhance usability, I added comprehensive client-side input handling with interactive prompts, clear error messages, and automatic flushing of excess input to prevent malformed messages, creating a robust and user-friendly chat experience.

C TCP Sockets

Dining Hall Meal Reviewing Platform

Together with a friend, I developed a full-stack web application that transforms the campus dining experience by enabling students to rank meals, write reviews, and engage socially through ratings, comments, and user follows.

I focused on the backend, writing extensive SQL and building the database architecture using PostgreSQL and Flask. I designed and optimized database schemas to support fast, scalable search and filtering across 10 dining halls, over 100 dishes, and user profiles, ensuring the platform could handle complex queries efficiently.
